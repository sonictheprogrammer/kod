# Kod
**Welcome!**

![scriptwelcome](https://github.com/sonictheprogrammer/kod/assets/92324146/4fdee8bf-0b70-4cad-96e1-7604b42c3389)


**kod** is a **git** project which is used to test if a **git project** works.

**kod's** name **originated** from the **translation** of the letter **Code**, Which is **Код**.

## Tutorial

### Gentoo Based Distributions

Before you can do anything else, Update your system.

```sh
emerge --ask --verbose --update --deep @world
```

If you don't have Git, install it by using emerge.

```sh
emerge --ask git
```

After that, clone the project,

```sh
git clone https://github.com/sonictheprogrammer/kod
```

Now, go into the files.

```sh
cd kod
```

Check the files!

```sh
ls
```

Now, unlock the files,

```sh
chmod +x git.sh
```

```sh
chmod +x run.sh
```

Now, Run it!

```sh
./git.sh
```

run.sh not running automatically? just run it manually!

```sh 
./run.sh
```

### Arch based Distributions
Update the system first,

```sh
sudo pacman -Syu
```

If you don't have git. You can install it by doing this command.

```sh
sudo pacman -S git
```

To get the project, clone the project.

```sh
git clone https://github.com/sonictheprogrammer/kod
```

Then, get into the files.

Before

```sh
user/~: cd kod
```

After

```sh
user/~kod:
```

Do this command to check if there are files.

```sh
ls
```

To prepare, use this command.

```sh
chmod +x git.sh
```

Also, to make sure that run.sh can work, use this command.

```sh
chmod +x run.sh
```

Then test the git project by doing this command!

```sh
./git.sh
```
